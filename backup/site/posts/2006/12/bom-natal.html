Bom Natal

Para o trabalho em part-time que é tirar fotos vestido de vermelho ao pé da criançada, para o Action Man e para a Barbie, para os sul americanos que estão na praia, para os que neste dia têm uma papa qualquer enviada pela ONU como jantar, para a Coca-Cola, para os criadores de perú, para as famílias que vêm na noite de 24 uma oportunidade para se juntarem, para as crianças, sem dúvida, para os que não pedem nada, para os que têm uma lista de quilómetros, para os que não oferecem nada, para os que gastam o subsídio de Natal realmente na quadra e para os que o deixam para a passagem de ano, para o almoço de 25, que são os restos do jantar de 24, para quem trabalha na data, para a união das pessoas sob aquele sentimento de quem não precisa de mais um "Bom Natal" que o silêncio fala por si, para as canções irritantes (o let it snow sofre-se), para os que enviam sms's às carradas e para todos os que fazem do Natal uma época melhor, embora quase "silly season", para quem nestes dias absorve alguma noção das palavras compreensão e tolerância. Para os bons corações e para quem não tem possibilidade de o viver como eu inserido numa sociedade dita moderna: Bom Natal!
