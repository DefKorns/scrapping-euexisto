Este País

Este país não é para velhos. Este país não é para novos. Este país não é para doentes. Este país não é para fumadores, nem é para condutores. Este país não é para trabalhadores. Não é para empresários. Este país não é para agricultores. Este país não é para empreendedores. Este país não é para professores. Este país não é para enfermeiros.Nem é para dentistas. Este país não é para criativos. Nem tão-pouco para encenadores, actores e escultores. Este país não é para optimistas. Nem mesmo para pessimistas. Este país não é para uns. Nem chega verdadeiramente a ser para outros. Este país não chega bem a ser um «país». Um problema que temos connosco mesmos.
